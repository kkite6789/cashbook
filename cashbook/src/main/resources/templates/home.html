1. 회원정보

1.1 회원 생성
-사진은 jpeg jpg png 파일 가능
-id 중복확인
1.2 회원 수정
-사진 변경시 원래 있던 사진파일 삭제 후 현재 사진으로 만듬
-id를 제외한 모든항목 수정가능
1.3 회원 삭제
-삭제하면서 사진파일도 삭제한다.
1.4 로그인
-세션값을 이용하여 id와 사진도 전송해 항상 사이드메뉴에 나오게 해준다.

2. 일일가계부

-월 부분을 누르면 해당 달의 월별가계부로 넘어간다.
-생성,수정,삭제
-생성은 선택되어 있는 날짜에 생성되게 설정

3. 월별가계부

-생성,수정,삭제
-금액을 누르면 해당 날짜로 일일가계부로 넘어간다.
-월 합계 기능 구현
-월별비교 누를시 해당 연도의 월별비교로 넘어감

4. 월별비교

-해당 연도의 총 합계 기능 구현
-금액을 누를시 월별가계부의 월로 넘어감

5.게시판

-글 생성,수정,삭제
-댓글 생성,수정,삭제
